# Movie App - Backend
Backend part of the Movie App application.

## Technologies used:
* NestJS + Typescript
* PostgreSQL + PrismaORM
  
## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# e2e tests
$ yarn run test:e2e
```
